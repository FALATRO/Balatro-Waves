# Welcome to the Wuthering Waves Balatro mod! 
This is a mod that focuses on adding references to Wuthering Waves into Balatro in the forms of Jokers, Boss Blinds, Challenges, and more!

* Note: This mod requires [Steamodded](https://github.com/Steamodded/smods?tab=readme-ov-file) and [Lovely](https://github.com/ethangreen-dev/lovely-injector).

## This mod as of now adds the following:

- 34 new Wuthering Waves themed Jokers
- 4 Wuthering Waves themed Boss Blinds
- 6 new Challenges
- 1 new Deck
- 2 new Vouchers
- 6 new Enhanced Cards
- 6 new Tarot Cards
- A new consumeable type
- 4 new Booster Packs
- 5 new Editions

## How to Download:
1. Install the latest version of Steamodded
2. Install the latest version of Lovely
3. Unzip download folder and put it in your mods folder
4. Enjoy!
